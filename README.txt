Bienvenido a mi portfolio de desarrollo.  
En este repositorio iré publicando los distintos proyectos personales que voy desarrollando para consolidar y demostrar mis conocimientos técnicos, principalmente en el ecosistema Java y Spring Boot, así como en tecnologías frontend y de integración habituales en proyectos reales.

El objetivo de este portfolio es mostrar código claro, estructurado y mantenible, aplicando buenas prácticas y patrones habituales en entornos profesionales.


Tecnologías principales

Backend
- Java
- Spring Boot
- Spring Data JPA / Hibernate
- APIs REST
- Seguridad con **JWT (JSON Web Token)**
- Control de roles y permisos
- Validaciones y manejo de errores

Frontend
- Angular
- TypeScript
- Consumo de APIs REST
- Separación de componentes y servicios

Documentación y testing
- Swagger / OpenAPI para documentación de APIs
- Postman
- Tests (unitarios y de integración, según proyecto)

Otros
- Bases de datos relacionales (MySQL / PostgreSQL)
- Git y control de versiones
- Docker (en algunos proyectos)
- Arquitectura en capas

Estructura del repositorio

Cada proyecto estará organizado en su propio directorio, incluyendo:

- Código fuente
- README específico del proyecto
- Instrucciones de instalación y ejecución (dentro del README)
- Ejemplos de uso (endpoints, capturas, etc.)
